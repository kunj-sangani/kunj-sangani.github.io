# Introduction

This blog helps in enabling multilingual features in SharePoint Communication Sites and creating multilingual modern pages.

# How to Enable Features

To enable the latest multilingual feature in communication site, follow the below steps

1. Navigate to SharePoint site collection settings
URL:- https://yoursite.sharepoint.com/_layouts/settings.aspx

2. Select the "Language settings" option present inside "Site Administration"

![placeholder](/assets/img/Blogs/Blog5/Image_1.png)

3. Enable the toggle button with the below text
"Enable pages and news to be translated into multiple languages"

4. From the dropdown "Select or type a language"
Either open the dropdown or type the language for which translations are required

5. We can specify translators if required this would send an email to the user when a translation for that specific language is available

![placeholder](/assets/img/Blogs/Blog5/Image_2.png)

# How to Create a Translation for a Specific Page

1. Navigate to the Page where we require a translation
2. Click on the "Translation" present as shown in the image below
3. To create a Translation for all the languages, click on "Create for all languages"
4. To create one for a specific language, click on "Create" next to the language

![placeholder](/assets/img/Blogs/Blog5/Image_3.png)

# How to Change the Language for the Page

1. In the top right corner, we get a dropdown which displays all available translations for the page
2. We can select any translations and can see the translated page.

![placeholder](/assets/img/Blogs/Blog5/Image_4.png)

# Translated Pages Location

1. A folder with the language initials is created in the "Site Pages" Library
2. All the translated pages are present inside the folder.

![placeholder](/assets/img/Blogs/Blog5/Image_5.png)
